# MANE 베타 테스트 가이드

MANE을 사용해주셔서 감사합니다! 이 가이드는 베타 테스터분들께 도움이 되도록 작성되었습니다.

---

## 🎯 MANE이란?

**MANE**은 1인~소규모 미용실을 위한 **고객 관리 시스템**입니다.

**주요 기능:**
- 고객 정보 관리
- 시술 이력 기록
- 예약 관리
- 매출 통계
- 미용사별 통계
- 엑셀 내보내기

---

## 📱 시작하기

### 1. 회원가입 및 로그인
1. 웹사이트 접속
2. 이메일 주소로 회원가입
3. 로그인

> 💡 **베타 테스터 혜택**: 모든 프리미엄 기능 무료 제공!

---

## 📖 주요 기능 사용법

### 1️⃣ 고객 관리

**고객 추가:**
1. 좌측 메뉴에서 **고객 관리** 클릭
2. 우측 상단 **추가** 버튼 클릭
3. 이름, 전화번호 입력 (메모 선택)
4. **추가** 버튼 클릭

**고객 검색:**
- 상단 검색창에 이름, 전화번호 입력

**단골 고객 필터:**
- **단골만** 버튼 클릭 → 3회 이상 방문 고객만 표시

**고객 정보 의미:**
- ⭐ **단골 배지**: 3회 이상 방문 고객
- 📅 **예약 알림**: "2일 후 예약" (파란색)
- 🟢 **재방문 예정**: "오늘 재방문 예정!" (초록색)
- 🟠 **재방문 지남**: "재방문 10일 지남" (주황색)
- 총 방문 횟수 표시

---

### 2️⃣ 시술 관리

**시술 메뉴 추가:**
1. **시술 관리** 페이지 이동
2. 시술 메뉴 섹션에서 **추가** 버튼
3. 시술 이름, 가격 입력
4. 담당 미용사 선택 (여러 명 가능, 선택 안 하면 모든 미용사)
5. **추가** 버튼 클릭

**미용사 추가:**
1. **시술 관리** 페이지 하단
2. 미용사 섹션에서 **추가** 버튼
3. 이름, 전화번호 입력
4. **추가** 버튼 클릭

> 💡 각 미용사는 자동으로 색상이 배정되어 캘린더에서 구분됩니다.

---

### 3️⃣ 예약 관리

**예약 추가:**
1. **예약 관리** 페이지 이동
2. 우측 상단 **추가** 버튼 또는 원하는 시간대 클릭
3. 고객, 날짜, 시간 선택
4. 담당 미용사 선택 (선택)
5. 시술 메뉴 선택 (선택)
6. 메모 입력 (선택)
7. **추가** 버튼 클릭

**예약 완료 처리:**
1. 예약 클릭
2. 상태를 **완료**로 변경
3. **수정** 버튼 클릭
4. → **자동으로 시술 이력에 추가됩니다!** ✨

**예약 취소:**
1. 예약 클릭
2. 상태를 **취소**로 변경
3. **수정** 버튼 클릭
4. → 시술 이력이 있었다면 자동 삭제됩니다.

**캘린더 특징:**
- 시간대별 예약 표시 (09:00 ~ 20:00)
- 같은 시간에 여러 예약 가능
- 미용사별 색상 구분 (좌측 색상 바)
- 완료/취소 상태 표시

---

### 4️⃣ 고객 상세 페이지

**접근 방법:**
- 고객 목록에서 고객 클릭

**시술 이력 추가:**
1. **시술 추가** 버튼 클릭
2. 날짜, 시술 메뉴, 가격 입력
3. 메모 입력 (선택)
4. **추가** 버튼 클릭

**사진 업로드:**
1. 시술 이력 클릭
2. **사진 추가** 버튼
3. 시술 전/후 사진 선택
4. 업로드

> 💡 시술 이력은 날짜 내림차순으로 정렬됩니다.

---

### 5️⃣ 통계

**통계 페이지:**
1. 좌측 메뉴에서 **통계** 클릭

**제공되는 통계:**
- **이번 달 매출**: 당월 총 매출
- **이번 달 방문**: 당월 시술 건수
- **단골 고객**: 3회 이상 방문 고객 수
- **평균 주기**: 고객들의 평균 재방문 간격 (일)
- **재방문율**: 2회 이상 방문한 고객 비율 (%)
- **월별 매출 차트**: 최근 6개월 매출 추이
- **월별 방문 횟수 차트**: 최근 6개월 방문 추이
- **미용사별 통계**: 각 미용사의 매출, 건수, 시술별 상세

---

### 6️⃣ 엑셀 내보내기

**통계 페이지에서:**
1. 상단에서 **기간 선택** (이번 달, 지난 달, 최근 3개월 등)
2. 원하는 엑셀 다운로드 버튼 클릭:
   - **시술기록**: 날짜, 고객명, 시술명, 금액, 메모
   - **매출통계**: 월별 매출, 건수, 평균 단가, 합계
   - **미용사통계**: 미용사별 요약 + 시술 상세 (2개 시트)

**고객 관리 페이지에서:**
1. 우측 상단 다운로드 아이콘 클릭
2. **고객 목록** 엑셀 다운로드
   - 이름, 전화번호, 메모, 최근 방문, 등록일

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 무료인가요?
A. 현재 베타 테스트 기간 동안 **모든 기능 무료**입니다.

### Q2. 데이터는 안전한가요?
A. Firebase(구글) 클라우드에 안전하게 저장됩니다. 사용자별로 데이터가 분리되어 다른 사람은 볼 수 없습니다.

### Q3. 모바일에서도 사용 가능한가요?
A. 네! 웹 브라우저로 접속하면 모바일에서도 사용 가능합니다.

### Q4. 예약을 완료로 바꿨는데 시술 이력에 안 보여요.
A. 예약에 시술 메뉴가 선택되어 있어야 시술 이력에 자동 추가됩니다.

### Q5. 고객 수 제한이 있나요?
A. 베타 테스터는 **무제한**입니다.

### Q6. 엑셀 파일이 열리지 않아요.
A. 엑셀, 구글 스프레드시트, Numbers 등 엑셀 호환 프로그램이 필요합니다.
